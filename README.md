# Advanced GitHub Copilot Workshop

An advanced workshop for experienced GitHub Copilot users looking to take their skills to the next level.

## Overview

This workshop covers advanced Copilot techniques including MCP (Model Context Protocol) server integrations, advanced prompting patterns, and deeper workflow optimizations.

## Structure

- **instructions/** – Documentation about the GitHub Copilot Instructions feature
- **mcp_servers/** – MCP server documentation and configuration
- **images/** – Supporting visual materials
