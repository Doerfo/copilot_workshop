# Advanced GitHub Copilot Workshop

An advanced workshop for experienced GitHub Copilot users looking to take their skills to the next level.

## Overview

This workshop is designed to help you master advanced GitHub Copilot features through a learn-by-doing approach. You'll explore powerful capabilities like MCP servers, custom instructions, prompts, agents, and the Copilot Specification Kit—then immediately put them into practice through hands-on tasks.

## Structure

- **knowledge/** – Learning materials covering core concepts:
  - GitHub Copilot fundamentals
  - MCP (Model Context Protocol) servers
  - Custom instructions
  - Advanced prompting techniques
  - Agent workflows
  - Copilot Specification Kit

- **tasks/** – Hands-on exercises to implement your learnings:
  - Repository setup
  - Instruction configuration
  - Prompt customization
  - Feature development with Spec Kit

- **local-docs/** – Supporting documentation and examples
- **images/** – Visual materials and diagrams
